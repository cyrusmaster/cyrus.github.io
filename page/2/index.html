QPlayer 迷你悬浮音乐播放器添加与配置

Posted on 2020-01-09 Edited on 2020-01-11 In 未分类

这两天都在想办法给博客添加一个音乐插件 试了很多 有的不太兼容 有的用不了 目前发现QPlayer这个还不错 因此分析一下如何安装及使用 插件github下载地址

安装后如图进行配置

点击添加歌曲列表 发现并不会解析真实地址

故此，想办法得到真实地址 如： http://music.163.com/#/song?id=317151 很明显，ID是317151 那么，这首歌的真实地址就是： http://music.163.com/song/media/outer/url?id=317151.mp3 这个地址不会失效的； 添加即可 参考地址： https://my.oschina.net/zhenggao/blog/3000778?device=geekTime.android http://www.360doc.com/content/18/0313/06/30724179_736535050.shtml

sh脚本如何自启 （rpi）

Posted on 2020-01-08 Edited on 2020-01-11 In 未分类

问题： 服务器每次开机都要手动启动sh脚本 过于繁琐 于是便查阅资料如何自启

查阅到其中一个简单的方法如下

修改 rc.local 文件 vi /etc/rc.local 第一次更改 以为 用户权限的问题 才导致自启失败 又进入root 排错 第二次更改 发现有的脚本没有启动 后来发脚本执行路径的问题 第三次更改

成功了！！

简单几步添加Live 2D

Posted on 2020-01-07 Edited on 2020-01-11 In 站点美化

1 header.php 中head标签添加如下内容：

<!-- 右下角live2d效果 -->
<script src="https://eqcn.ajz.miesnfu.com/wp-content/plugins/wp-3d-pony/live2dw/lib/L2Dwidget.min.js"></script>

<script>
    L2Dwidget.init({
        "model": {
            jsonPath: "https://unpkg.com/live2d-widget-model-shizuku@1.0.5/assets/shizuku.model.json",
            "scale": 1
        },
        "display": {
            "position": "left",
            "width": 150,
            "height": 300,
            "hOffset": 0,
            "vOffset": -20
        },
        "mobile": {
            "show": true,
            "scale": 0.5
        },
        "react": {
            "opacityDefault": 0.7,
            "opacityOnHover": 0.2
        }
    });
</script>

<!--
其他可选的模型：
黑猫：https://unpkg.com/live2d-widget-model-hijiki@1.0.5/assets/hijiki.model.json
萌娘：https://unpkg.com/live2d-widget-model-shizuku@1.0.5/assets/shizuku.model.json
白猫：https://unpkg.com/live2d-widget-model-tororo@1.0.5/assets/tororo.model.json
狗狗：https://unpkg.com/live2d-widget-model-wanko@1.0.5/assets/wanko.model.json
